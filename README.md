# localio

Projet Tutoré: formation MIW

# Lancer le projet :

`ddev start`

`ddev composer install`

`npm install`

`ddev exec php artisan migrate`

`ddev exec php artisan db:seed`

# Appliquer les styles :

`npm run dev`

# Passer en compte admin :

`ddev sequelace`

TABLE 'users', COLONE 'role_id', passer la valeur à `4`
